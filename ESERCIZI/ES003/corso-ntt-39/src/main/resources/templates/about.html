Panoramica del Progetto

Questo progetto è stato sviluppato nell'ambito del Corso NTT DevOps ed è un'applicazione web completa che dimostra l'integrazione di diverse tecnologie moderne per lo sviluppo di applicazioni enterprise.

L'applicazione implementa un sistema di autenticazione sicuro, aree pubbliche e private, e utilizza le best practices per lo sviluppo di applicazioni Spring Boot.

Stack Tecnologico
Backend
  • Java 21
  • Spring Boot 4.0.4
  • Spring MVC
  • Spring Security 6
  • Thymeleaf Template Engine
Frontend
  • HTML5
  • CSS3
  • Bootstrap 5.3
  • Bootstrap Icons
  • JavaScript
Funzionalità Implementate
  • Sistema di Autenticazione: Login sicuro con Spring Security
  • Gestione Utenti: Utenti in-memory con ruoli (USER, ADMIN)
  • Password Encoding: Sicurezza con BCrypt
  • Aree Pubbliche/Private: Separazione degli accessi
  • Design Responsive: Ottimizzato per tutti i dispositivi
DevOps Best Practices

Il progetto segue le best practices DevOps:

  • Architettura MVC ben strutturata
  • Separazione delle responsabilità (Controller, Service, Config)
  • Logging appropriato per monitoring
  • Gestione sicura delle credenziali
  • Codice ben commentato e documentato
  • Pronto per containerizzazione (Docker)
  • Pronto per orchestrazione (Kubernetes)
  • Integrabile con pipeline CI/CD (Jenkins)
